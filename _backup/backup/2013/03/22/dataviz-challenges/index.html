Lectures & Workshops

Dataviz Challenges

March 22, 2013 Sebastian Meier

This lecture was given in cooperation with Marcus Paeschke in the Interface Design department at the university for applied sciences Potsdam, during the summer of 2013.

The course was introducing the students to new methods and tools for data visualization. Throughout the term the students were given data sets ranging from meteorite data to data on crowdfunding in germany. The students had round about three weeks for every data set to work on. Below are a few examples from those challenges.

Student-Projects

  • Code Red DNA

    Visualization of the diffusion of the Code Red computer virus. Showing correlations in time and location.

    Project by Christopher Pietsch | Projectwebsite

  • Meteorite Media Impact

    Based on the visualising.org challenge to visualize a global dataset on meteorites, this project is comparing the media impact with the physical impact.

    Project by Christopher Pietsch and Luis Grass
    Projectwebsite

  • Meteor Impact

    The amount of meteor sightings and found meteorites has been rising since the middle of the 20st century. Spectacular events connected to meteorites happen aplenty. The media report – and fiction is inspired. We looked into this and collected stories from movies and books to examine them when juxtaposed with real meteorite discoveries.

    Project by Sören Knöll, Gregorio Gasperi and Christian Groß
    Projectwebsite, Project documentation

  • Meteorites 1900-2000

    Visualisation of all fell and found meteorites between 1900 & 2000. The graphic gives a special view on meteorite findings by comparing time and longitude on a radial axis. This kind of visualisation shows the human bias on the physical locations of found meteorites.

    Project by Kim Albrecht | Projectwebsite

  • Fußball & Glück

    The visualization shows the outcome of games of each team in the german football league. The direction shows a positive or a negative outcome the length of each stroke visualizes the goal difference..

    Project by Kim Albrecht | Projectwebsite

  • Faceted Search Interface

    This project is analyzing the existing interface of the Meteoritical Bulletin Database and trying to develop a new interface based on the idea of faceted search.

    Project by Simon Martin | Projectwebsite

  • Fireball from Outer Space

    This visualization is taking data from the Meteoritical Society. Through crossfilters and geographic visualizations the user can visually explore the dataset.

    Project by Sebastian Sadowski | Projectwebsite

  • Der Effizienzvergleich

    This visualization is taking a different point of view on the statistics of success in the german football league. It analyzes the correlation between money spend and points made.

    Project by Simon Martin and Sören Knöll
    Projectwebsite

  • Visualizing Meteorites

    The visualization shows the usual classification of meteorites, which is based on the origin of the meteorites parent body. A distinction is made between differentiated and undifferentiated meteorites.

    Project by Paul Vollmer | Projectwebsite

  • They came from outer space

    This project is using the global data set from the Meteoritical Society and turning it into a playful experience to learn more about meteorites.

    Project by Gabriel Credico | Projectwebsite

  • Money from Outer Space

    This project is visualizing meteorite data and correlating it with the price for meteorite material, showing that the most expensive ones are also the rarest ones.

    Project by Christoph Wiedenmann | Projectwebsite

  • Warst du schonmal in der Heimat deines Nachbarn?

    This project is visualizing the nationalities berlin is made up of and the most popular travel destinations. Visually exploring if people in Berlin get to know the countries of their neighbors are coming from.

    Project by Paul Heinicker, Sebastian Sadowski and Tim Pulver
    Projectwebsite 

  • Wer kauft die Bundesliga leer?

    This project is visualizing the transfer market of the german football league. Showing the trades between clubs especially the number of trades and the money spend.

    Project by Paul Heinicker, Sebastian Sadowski and Tim Pulver
    Projectwebsite
